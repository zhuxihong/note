```css
固定定位（参照窗口定位，不跟随网页滚动而滚动）
操作：上下左右的值，
定位特点：不脱离默认布局，会在原始位置保留元素的位置。适合对页面元素位置微调。绝对定位脱离文档流，不在默认布局中占据位置。通常在大范围元素内调整小元素的样式时使用。
图层：z-index:数字 数值越大越考上
```

### JavaScript:

```javascript
※ 变量声明：

JavaScript执行前会检查当前作用域所有声明的变量和函数，声明提前，值在顺序执行时再依次赋值。

- var全局变量 
- let局部变量 
- const 常量声明
----------------------------------------------------
※ 数据类型：

1. number 数值类型 
2. string  字符串类型   格式化字符串`hello ${变量名}`
3. boolean 布尔类型
4. undefined 为赋值（系统返回）
5. null 空（主动使用）
----------------------------------------------------

※ 数据类型转换：
转换字符串：数据.tosrting   
转换数字类型：Number（数据）
转换数字类型：parseInt（数据） 解析到无法解析的内容结束，返回解析到的内容.用于可取整
解析浮点数：parsefloat（数据） 能识别第一次出现的浮点数
----------------------------------------------------

※ 隐性转换：
字符串加其他类型，会把其他类型转换成字符串拼接；
字符串减数字类型，会把字符串转换成数字类型，做减法；
可以运用这个特性，去方便的转换数据类型；如果 123+""='123'   '123'-0=123
----------------------------------------------------

※ 【运算符】
++ -- 先返回最开始的数，再自加1
< > ==  比较符，会把值转换成数值笔记。特殊的：null==undfinded
=== 全等，类型和数值都一直
----------------------------------------------------

※ 逻辑运算符：
&&与  前面满足后面继续运行，前面不满足后面则不需要执行
||或  前不为真会继续执行后面的，有一处为真则为真
! 非  
----------------------------------------------------

※【三目运算】：
表达式1 ?表达式2:表达3
```
